Learn how to code

July 18th 2015

I will cover the basics of learning how to code. I personally went through all these steps and learnt a variety of things. The progress will be slow but you will reap your success later on.

The Basics

First thing you need to do is get accustomed to seeing code. Go to thenewboston site and start with the Java tutorial. Java is quite a simple programming language. Some of you might be wondering why I am suporting Java as a first language. There are a few reasons for that. Java syntax is quite simple (almost like normal english). There a uncountable amounts of resources made available by people to learn Java. Java also has very good IDEs and debugging tools (that are highly visual).

After finishing the beginner course, just carry on with the advance course. I can personally vouch for theNewBoston tutorials as I learnt most of my stuff from a huge 1000 page textbook (yea I am a book lover) and found that the tutorials covered most of it in a very short time. If you are still not satisfied then I recommend reading "Java - How to program" by Deitel. Its a lengthy book but extremely easy to understand even if you have never coded before.
